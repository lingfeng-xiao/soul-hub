# AI Agent 记忆系统设计 - 精简笔记

## 核心要点

### 1. 记忆分层架构
- **短期记忆**：当前会话上下文 (memory/*.md)
- **长期记忆**：持久核心 (MEMORY.md)
- **经验知识**：可复用模板 (notes/*.md)
- **系统配置**：运行依据 (AGENTS.md, TOOLS.md)

### 2. 记忆访问模式
- **主动回忆**：memory_search 语义搜索
- **被动注入**：每次启动时加载 USER.md、SOUL.md
- **增量更新**：学到的教训立即写入

### 3. 关键设计原则
- 文字 > 脑力：记住就写下来
- 去重检查：同一来源间隔2次以上才能重复
- 影响评估：高风险操作需先请示用户

### 4. 应用场景
- 跨会话保持上下文
- 避免重复提问
- 积累可复用经验
