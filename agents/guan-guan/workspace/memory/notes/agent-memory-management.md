# Agent 内存管理最佳实践

## 核心要点

### 1. 分层记忆架构
- **短期记忆**：会话级上下文，仅保留当前任务相关信息
- **长期记忆**：持久化存储核心知识、偏好、经验
- **工作记忆**：当前任务执行过程中的临时状态

### 2. 记忆压缩策略
- 定期摘要：将长对话压缩为关键要点
- 重要度筛选：保留高价值信息，丢弃冗余
- 向量索引：支持快速检索历史相关内容

### 3. 记忆读写原则
- **写入时机**：任务完成、关键决策点、用户偏好变化
- **读取策略**：基于当前任务上下文检索相关记忆
- **遗忘机制**：设置记忆过期策略，避免信息膨胀

### 4.实践经验
- OpenClaw 采用 memory/*.md 文件存储长期记忆
- 使用 memory_search 进行语义检索
- 通过 memory_get 精确提取所需记忆

## 应用建议
为管管添加记忆分层：
1. 短期：当前会话状态
2. 长期：MEMORY.md + notes/
3. 经验：AGENTS.md / TOOLS.md
