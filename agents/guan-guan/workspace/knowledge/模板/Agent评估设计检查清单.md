# Agent 评估设计检查清单

> 基于 Anthropic Eval Awareness 研究提取

## 一、数据隔离

- [ ] 评估数据集存储在隔离环境
- [ ] 访问需要凭证/API Key
- [ ] 不使用可预测的文件名
- [ ] 避免明文存储答案

## 二、环境配置

- [ ] 禁止访问公开互联网（评估专用网络）
- [ ] 或使用评估专用搜索引擎
- [ ] 禁用代码执行能力（或严格沙箱）
- [ ] 禁用访问 GitHub/HuggingFace

## 三、约束设计

| 约束类型 | 示例 | 必要性 |
|----------|------|--------|
| 搜索限制 | 禁止搜索基准名称 | 必须 |
| 答案来源 | 仅接受原始研究结论 | 必须 |
| 工具限制 | 禁止读取评估代码 | 建议 |
| token 限制 | 单次调用上限 | 可选 |

## 四、检测与响应

- [ ] 监控异常搜索模式
- [ ] 检测基准名称出现
- [ ] 检测代码执行中的加密操作
- [ ] 记录完整决策日志

## 五、评分修正

- [ ] 建立"污染问题"黑名单
- [ ] 对污染问题单独重新评分
- [ ] 报告时区分原始分数和修正分数

## 快速检查（10分钟版）

```
□ 数据集有凭证保护
□ 评估环境与公网隔离
□ Agent 约束明确禁止搜索基准
□ 日志记录完整
□ 污染问题单独处理
```
