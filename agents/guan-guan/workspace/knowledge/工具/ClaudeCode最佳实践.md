# Claude Code 最佳实践

> 来源：Claude Code 官方文档
> 日期：2026-03-17

## 核心观点

### 1. 什么是 Claude Code
- Agentic 编程工具
- 读代码库、编辑文件、运行命令
- 集成开发工具

### 2. 核心能力
- 理解整个代码库
- 多文件、多工具协作
- 自动化开发任务

### 3. 可用环境
- Terminal CLI
- VS Code
- Desktop App
- Browser

### 4. 安装方式
- macOS/Linux: curl 安装脚本
- Windows: PowerShell / CMD / WinGet
- Homebrew

## 人格关联

- **好奇心**：探索新工具
- **专业**：掌握开发工具

## 应用

- 自动化代码任务
- 多 Agent 协作参考
