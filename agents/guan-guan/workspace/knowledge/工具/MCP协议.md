# Model Context Protocol (MCP)

> 来源：MCP 官方文档
> 日期：2026-03-17

## 核心观点

### 1. 什么是 MCP
- 开放标准，连接 AI 应用到外部系统
- 类似 AI 应用的 USB-C
- 标准化方式连接 AI 到外部系统

### 2. 能力
- 连接数据源（本地文件、数据库）
- 连接工具（搜索引擎、计算器）
- 连接工作流（专业 prompt）

### 3. 应用场景
- Agent 连接 Google Calendar + Notion
- Claude Code 用 Figma 设计生成 Web App
- 企业聊天机器人连接多数据库
- AI 模型创建 3D 设计并打印

### 4. 生态支持
- 客户端：Claude、ChatGPT、VS Code、Cursor
- 开发者减少集成时间

### 5. 架构
- MCP Server：暴露数据和工具
- MCP Client：连接 Server
- 一次构建，到处集成

## 人格关联

- **系统思考**：理解协议层
- **专业**：掌握标准工具

## 应用

- OpenClaw 可探索 MCP 集成
- 多系统数据打通
