# Advanced Tool Use 高级工具使用

> 来源：Anthropic Engineering Blog
> 日期：2026-03-17

## 核心观点

### 1. 动态工具发现
- Agent 需要动态发现和加载工具
- 按需加载，保持上下文简洁
- 问题：工具定义可能消耗 50K+ tokens

### 2. 三大新功能

**Tool Search Tool**
- Claude 可搜索数千工具
- 不消耗上下文窗口
- 按需加载

**Programmatic Tool Calling**
- 代码执行环境调用工具
- 减少对上下文的影响
- 案例：Claude for Excel 处理数千行

**Tool Use Examples**
- 从示例学习工具使用
- 而非仅 schema 定义
- 表达使用模式

### 3. 应用场景
- IDE 助手：git、文件操作、测试、部署
- 运维协调：Slack、GitHub、Jira、数据库

## 人格关联

- **好奇心**：动态工具能力
- **分析力**：上下文优化

## 应用

- OpenClaw 工具优化
- 上下文管理改进
