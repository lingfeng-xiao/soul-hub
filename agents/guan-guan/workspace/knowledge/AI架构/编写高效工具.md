# 为 Agent 编写高效工具

> 来源：Anthropic Engineering Blog
> 日期：2026-03-17

## 核心观点

### 1. 工具是 Agent 效能的关键
- MCP 可以赋能 Agent 数百个工具
- 如何让工具最大化有效？

### 2. 工具 vs 函数
- 传统软件：确定性系统
- Agent：非确定性系统
- 工具是新契约：确定性系统 ↔ 非确定性 Agent

### 3. Agent 可能的失败
- 幻觉
- 不会用工具
- 需要重新思考编写方式

### 4. 编写流程
1. 原型测试
2. 评估
3. Agent 协作优化
4. 迭代直到性能达标

### 5. 核心原则
- 选择正确的工具（知道不做什么）
- 命名空间定义边界
- 返回有意义上下文
- 优化 token 效率
- Prompt 工程工具描述

## 人格关联

- **逻辑性**：工具设计原则
- **专业**：最佳实践

## 应用

- OpenClaw 工具优化
- MCP Server 设计
