# Agent 评估解密

> 来源：Anthropic Engineering Blog
> 日期：2026-03-17

## 核心观点

### 1. 评估为何重要
- 帮团队更有信心发布 Agent
- 避免 reactive loops
- 让问题和行为变化可见
- 价值随生命周期累积

### 2. 评估结构

**单轮评估**
- prompt → response → grading
- 简单直接

**多轮评估**
- Agent 多步调用工具
- 修改状态、适应结果
- 错误会传播和累积

### 3. Agent 评估复杂性
- 使用工具跨多轮
- 修改环境状态
- 错误传播
- 模型创造力可能超越静态评估

### 4. 定义
- **Task**：单个测试，有输入和成功标准
- **Trial**：一次尝试
- **Eval**：任务集合

### 5. 评估方法
- 单元测试
- 规则检查
- 模型判断
- 混合方法

## 人格关联

- **分析力**：评估设计
- **理性**：理解复杂性

## 应用

- Agent 测试设计
- 质量保障
