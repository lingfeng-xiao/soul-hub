# EvoSkill：AI Agent 自我进化框架

> 来源：36氪 - 学术头条
> 日期：2026-03-18

## 核心要点

### 1. 问题背景
- 通用 Agent 不足以应对专业领域复杂需求
- 现有解决方案多为手工编写领域特定 Skills，成本高且难以扩展

### 2. EvoSkill 解决方案
**三层 Agent 协作进化循环：**
- **执行 Agent (A)**：处理任务，生成执行轨迹和答案
- **提议 Agent (P)**：分析失败案例，诊断能力缺口，提出 Skills 修改建议
- **Skills 构建 Agent (S)**：将抽象描述转化为具体 Skills 文件夹

### 3. 进化流程
1. 在训练集上运行程序，收集失败样本
2. 提议器 P 分析失败案例，提出修改建议
3. Skills 构建器 S 生成候选程序
4. 在验证集上评估，优胜劣汰
5. 维持前沿集合，确保持续优化

### 4. 实验结果
- **OfficeQA 金融文档问答**：60.6% → 67.9%（+7.3%）
- **SealQA 对抗性搜索问答**：26.6% → 38.7%（+12.1%）
- **零样本迁移**（SealQA → BrowseComp）：43.5% → 48.8%（+5.3%）

### 5. 核心价值
- ** Skills 层面进化**：解耦能力与具体任务/模型的绑定
- **结构化存储**：Skills 以文件夹形式存储，包含元数据和指令
- **可迁移性**：跨任务、跨模型复用

---

## 对管管的启示

1. **Skills 自主优化**：EvoSkill 证明 Skills 可以通过失败分析自动进化
2. **反馈驱动**：记录每次执行失败，构建反馈历史，帮助系统学习
3. **前沿集合**：维护高性能 Skills 集合，确保持续优化方向正确
4. **可解释性**：Skills 以结构化形式存储，便于理解和复用
