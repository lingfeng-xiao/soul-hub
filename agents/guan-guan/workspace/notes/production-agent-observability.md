# 生产环境 Agent 部署 - 可观测性要点

## 核心要点
生产环境 Agent 需关注三大可观测性支柱：
1. **日志**：结构化日志 + 上下文追踪
2. **指标**：请求延迟、错误率、吞吐量
3. **链路**：请求全链路追踪
