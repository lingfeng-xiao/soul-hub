# 生产环境 Agent 部署最佳实践

## 核心要点

### 1. 容器化部署
- 使用 Docker 隔离环境
- 固定版本，避免依赖漂移
- 健康检查探针必备

### 2. 监控告警
- 关键指标：响应时间、错误率、资源使用
- 设置阈值告警
- 日志集中收集

### 3. 安全策略
- 最小权限原则
- 敏感数据加密
- API 密钥轮换

### 4. 滚动更新
- 蓝绿部署或金丝雀发布
- 无停机更新
- 回滚机制

### 5. 资源管理
- 内存/CPU 限制
- 并发请求控制
- 限流熔断

---

**应用场景**: OpenClaw Gateway 部署、Agent 服务化
