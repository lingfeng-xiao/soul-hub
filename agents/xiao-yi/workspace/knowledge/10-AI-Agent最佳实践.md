# AI Agent 最佳实践

> 来源：小艺学习产出
> 标签：AI、Agent、自动化

## 什么是 AI Agent？

能自主规划、执行、迭代的 AI 系统。

## 核心能力

| 能力 | 说明 |
|------|------|
| 感知 | 理解环境、获取信息 |
| 推理 | 分析问题、制定计划 |
| 执行 | 调用工具、完成行动 |
| 迭代 | 从结果中学习改进 |

## 设计模式

### 1. ReAct (Reason + Act)
思考 → 行动 → 观察结果 → 迭代

### 2. Tool Use
- 明确工具能力边界
- 精确描述工具输入输出
- 错误处理和重试

### 3. Planning
- 分解复杂任务为子任务
- 按依赖关系排序
- 逐步验证

## OpenClaw 实践

- 自然语言描述需求
- 自动查找并执行方法
- 7×24 小时运行

## 关键原则

1. **最小化人工干预**
2. **清晰的反馈机制**
3. **安全的工具边界**
