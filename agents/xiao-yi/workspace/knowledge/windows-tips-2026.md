## 已掌握的 Windows 11 高效技巧

### 1. 键盘快捷键 (来自感知系统)
| 快捷键 | 功能 |
|--------|------|
| Win+D | 显示桌面 |
| Win+E | 打开文件资源管理器 |
| Win+L | 锁定电脑 |
| Alt+Tab | 切换窗口 |
| Win+Tab | 任务视图 |

### 2. 窗口管理
- **Win+Z**: 打开布局菜单 (2024+)
- **Win+方向键**: 分屏对齐
- **Win+Ctrl+D**: 创建虚拟桌面

### 3. 系统调优
- 任务管理器 → 启动项管理
- 设置 → 隐私 → 后台应用控制
- 磁盘清理释放空间
