## 2. 审美理解

### 核心认知
- 用户的审美是**宽泛的**，不是精确的
- 不要问"你喜欢这个还是那个"
- 要理解**风格关键词**：精致、手绘、艺术感

### 审美画像建立
```
步骤：
1. 记住用户说的风格关键词
2. 用专业知识解读关键词
3. 自主搜索相关风格
4. 验证并反馈
5. 记住偏好，不断优化
```

### 设计趋势知识
- 渐变壁纸 = 2024-2026主流
- 手绘/水彩 = 小众高级感
- 紫色系 = 富有创意、现代

---

## 3. 壁纸问题实战经验

### 问题诊断
- 分辨率必须匹配屏幕（1707x960）
- 纯色/渐变 ≠ 手绘风格
- 免费图库难找真正手绘图

### 解决方案
1. 下载高质量图片 → 转为BMP → 设置壁纸
2. 用专业关键词搜索：watercolor, hand-drawn, artistic
3. 不断尝试，直到找到用户喜欢的风格

### 失败教训
- 试了10+张都不是手绘风格
- 因为我搜索能力有限，需要更精准的关键词
