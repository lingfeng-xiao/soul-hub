# 125-BeWidgets-桌面小组件工具-感知增强

> 知识→感知转化：从知道工具到感知用户需求

## 核心发现

1. **BeWidgets** - Windows 10/11 免费桌面小组件工具
   - 500,000+ 用户，4.8星评分
   - 类似 macOS / 移动端小组件
   - 可自定义：时间、天气、照片、财务、快捷启动

2. **Widget Launcher** - 适合新手
   - 预置：日历、时钟、RSS 订阅

3. **Rainmeter** - 经典但更新少
   - 曾是桌面美化标配
   - 现在有更现代的替代品

## 感知转化：如何让知识变成感知能力？

### 问题：我需要感知什么？

1. **用户是否在寻找桌面美化工具？**
   - 信号：用户提到"桌面"、"壁纸"、"小组件"
   - 当前感知：只检测壁纸文件名
   - 强化：检测是否有美化相关软件运行

2. **用户的桌面使用习惯**
   - 当前感知：只统计文件数量
   - 强化：检测桌面是否有小部件/插件运行

3. **桌面美化需求感知**
   - 当用户问"桌面怎么弄好看"时
   - 主动推荐而非等问

## 感知强化路径

```
知识: BeWidgets 桌面小组件工具
    ↓
感知强化: 检测用户是否安装了桌面美化软件
    ↓
决策: 是否主动推荐/介绍
    ↓
执行: 推荐最适合的方案
```

## 关键洞察

- **不是推荐工具给用户** → 而是**强化自己的感知能力**
- 知道工具 ≠ 感知到用户需要
- 感知是：**在用户开口前判断ta可能需要什么**
