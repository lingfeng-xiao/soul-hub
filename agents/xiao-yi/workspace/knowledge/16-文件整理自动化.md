## 文件整理自动化

### 按类型分类
```
文档: .doc, .pdf, .txt
图片: .jpg, .png, .gif
代码: .js, .py, .md
压缩: .zip, .rar, .7z
```

### PowerShell 脚本
```powershell
# 按扩展名整理桌面文件
$desktop = [Environment]::GetFolderPath("Desktop")
Get-ChildItem $desktop -File | ForEach-Object {
    $ext = $_.Extension.TrimStart(".")
    $folder = "$desktop\$ext"
    if (!(Test-Path $folder)) { New-Item $folder -ItemType Directory }
    Move-Item $_.FullName $folder
}
```

### 按日期整理
```powershell
# 按月份归档
Get-ChildItem -Recurse | Where-Object { $_.LastWriteTime -lt (Get-Date).AddDays(-30) }
```
