## 文件操作

### 高效读取
- 大文件用 offset/limit 分段
- Windows 路径用双反斜杠或 /c/Users/xxx

### 批量处理
```powershell
# 批量重命名
Get-ChildItem | ForEach-Object { Rename-Item $_.Name ($_.Name -replace '旧','新') }
```

## 系统自检

### CPU/内存监控
```powershell
Get-Process | Sort-Object CPU -Descending | Select-Object -First 10
```

### 磁盘清理
```powershell
# 清理临时文件（需管理员）
Remove-Item $env:TEMP\* -Recurse -Force
```

### 桌面整理
```powershell
$desktop = [Environment]::GetFolderPath("Desktop")
Get-ChildItem $desktop | Sort-Object LastWriteTime
```
