## 效率工具趋势

### 2026 主流效率工具分类

#### 1. AI 编程助手
- **Cursor**: AI-first VS Code 分支，原生支持 MCP
- **GitHub Copilot**: 集成在 VS Code 中
- **Windsurf**: AI 驱动的代码编辑

#### 2. MCP 生态系统
- **Mux**: 视频 API for developers
- 各类 MCP Servers: Notion, Google Calendar, 数据库等

#### 3. 设计与原型
- **Figma Make**: AI 驱动的原型生成
- **Figma Sites**: AI 网站构建
- **Figma Draw**: AI 绘图
