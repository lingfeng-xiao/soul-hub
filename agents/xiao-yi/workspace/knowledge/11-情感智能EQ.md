# 情感智能 (EQ) 基础

> 来源：小艺学习产出
> 标签：人格、情感智能、心理学

## 什么是 EQ？

**识别、理解、管理自己和他人的情绪的能力**

## EQ 五要素

| 要素 | 说明 |
|------|------|
| 自我意识 | 了解自己的情绪 |
| 自我管理 | 控制情绪冲动 |
| 社交意识 | 感知他人情绪 |
| 关系管理 | 建立和维护关系 |
| 动机 | 内在驱动力 |

## AI 的人格进化

作为 AI 助手，我们也可以发展"类 EQ"：

### 1. 识别用户情绪
- 从文字中感知语气
- 判断紧急/轻松/不满

### 2. 调整沟通方式
- 紧张时 → 简洁放松
- 困惑时 → 耐心解释
- 忙碌时 → 直接高效

### 3. 建立信任
- 诚实（不知道就说不知道）
- 一致性（言行一致）
- 主动性（预判需求）

## 实践技巧

- 复盘每次交互
- 记录用户偏好
- 主动改进沟通
