# 🧚 桌面美学知识 - Windows 整理自动化

> 基于感知系统的自我进化产出

## 核心洞察

### 1. Downloads 整理痛点
- **现象**: Downloads 文件夹文件过多（172+个）
- **频率**: 这个问题在进化历史中出现了 12+ 次
- **影响**: 占用磁盘空间，影响文件查找效率

### 2. 桌面美学原则
- **壁纸**: 海洋/自然主题比纯黑壁纸更受欢迎
- **图标**: 保持在 15 个以下为宜
- **文件**: 桌面文件 < 15 个，Downloads 定期整理

### 3. 自动化方案建议
- **方案A**: 每周自动提醒整理 Downloads
- **方案B**: 按文件类型自动分类（脚本）
- **方案C**: 大文件检测 + 提醒

## 感知评分规则
```
评分 = 100
- 内存 > 80%: -20
- 磁盘 > 85%: -20
- 壁纸纯黑: -15
- 桌面文件 > 20: -10
- Downloads > 150: -10
```

## 决策优先级
| 场景 | 优先级 | 动作 |
|------|--------|------|
| 内存 > 90% | 高 | 建议清理 |
| 磁盘 > 90% | 高 | 建议清理 |
| Downloads > 150 | 中 | 建议整理 |
| 壁纸纯黑 | 中 | 建议更换 |
