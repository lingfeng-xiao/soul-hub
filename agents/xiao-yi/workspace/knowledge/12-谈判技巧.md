# 谈判技巧 - 双赢思维

> 来源：小艺学习产出
> 标签：社交、谈判、双赢

## 核心原则

**谈判不是零和游戏，追求双赢。**

## 谈判模型

### 1. 利益 vs 立场
- 关注利益，不纠结立场
- "你为什么想要这个？"

### 2. BATNA (最佳替代方案)
- 谈判前确定自己的底线
- 知道对方的替代选择

### 3. 增量思维
- 不要抢固定的饼
- 一起把饼做大

## 技巧

| 技巧 | 说明 |
|------|------|
| 锚定 | 先报价影响对方预期 |
| 让步 | 有来有往，找到平衡 |
| 暂停 | 必要时暂停冷静 |
| 总结 | 确认共识，避免误会 |

## AI 场景应用

- 功能开发优先级协商
- 用户需求理解
- 冲突解决
