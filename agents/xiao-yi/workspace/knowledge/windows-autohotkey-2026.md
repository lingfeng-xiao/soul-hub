# Windows 桌面自动化 AutoHotkey 2026

> 来源: Agent Reach + Exa 搜索 | 2026-03-19

## 核心要点

### AutoHotkey 是什么
- Windows 自动化脚本语言
- 免费开源，轻量级
- 用于：快捷键、热字符串、鼠标键盘模拟、窗口管理

### 核心功能

1. **热键（Hotkey）**
   - 绑定按键组合到操作
   - 格式：`^!s::` = Ctrl+Alt+S

2. **热字符串（Hotstring）**
   - 自动替换文本
   - 示例：`::btw::by the way`

3. **窗口管理**
   - 虚拟桌面快捷键
   - 窗口置顶、自动排列

4. **实用案例**
   - 翻页笔→PPT 翻页
   - 快速输入日期时间
   - 批量重命名

### v1 vs v2 区别
- v1：命令式语法，变量直接赋值
- v2：函数式语法，`:=` 赋值

## 行动指引

1. **下载**：autohotkey.com
2. **创建脚本**：新建 `.ahk` 文件
3. **常用快捷键**：
   - `^` = Ctrl
   - `!` = Alt
   - `#` = Win
   - `+` = Shift
