## 核心要点

### PowerToys 是什么
- 微软官方 Windows 系统增强工具集
- 15+ 实用功能，完全免费开源
- 提升电脑操作效率 300%

### 核心功能

1. **FancyZones** - 窗口管理大师
   - 一键将窗口吸附到自定义网格
   - 支持保存多种布局方案
   - 游戏、办公、编程快速切换

2. **PowerRename** - 批量重命名神器
   - 支持正则表达式
   - 可预览修改效果
   - 还能撤销操作

3. **ColorPicker** - 屏幕取色专家
   - 快捷键 `Win+Shift+C`
   - 支持 RGB、HEX、HSV 格式

4. **TextExtractor** - 文字提取魔术
   - 快捷键 `Win+Shift+T`
   - 截图自动识别文字
   - 准确率 98%

5. **Command Palette** - 命令面板
   - 类似 macOS Spotlight
   - 可自定义背景图和颜色
   - 可调整分类优先级

## 行动指引

1. **下载安装**：Microsoft Store 或 GitHub
2. **常用快捷键**：
   - `Win+Shift+C` 取色
   - `Win+Shift+T` 识字
   - `Win+Shift+Z` 窗口布局
3. **根据场景创建布局方案**
