# 飞书 API 整合手册

> 小艺的知识库 - 持续更新

## 可用工具

| 工具 | 功能 |
|------|------|
| feishu_calendar_* | 日程管理 |
| feishu_task_* | 任务管理 |
| feishu_bitable_* | 多维表格 |
| feishu_doc_* | 文档操作 |
| feishu_im_* | 消息收发 |

## 关键注意事项

### 日程创建
- 必须传 user_open_id，否则用户看不到
- 时间用 ISO 8601 格式

### 消息发送
- 需要用户授权或好友关系
- content 必须是合法 JSON

### 多维表格
- 字段类型：1=文本，2=数字，3=单选...
- create 用 fields，batch_create 用 records

## 典型场景

### 自动日报
1. 定时触发（cron）
2. 汇总数据
3. 发送到飞书

### 任务提醒
1. 监听任务到期
2. 提前通知
3. 记录完成状态
