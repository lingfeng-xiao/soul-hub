## 知识要点（从搜索中提取）

### 1. 高RAM使用的根因
- 启动项（Startup programs）
- 后台进程（Background processes）
- 过期驱动（Outdated drivers）
- 恶意软件/内存泄漏（Malware/Memory leaks）
- Windows 11高级功能消耗（VBS、虚拟化平台）

### 2. 快速排查方法
1. 任务管理器 → 内存排序 → 找异常进程
2. 启动管理 → 禁用不需要的启动项
3. 服务管理 → 禁用非必要后台服务

### 3. 可自动化感知的方向
- **进程画像**：记录高频高内存进程（如node.exe）
- **趋势检测**：不只是当前值，而是上升趋势
- **启动项感知**：检测新增启动项
