# Windows 11 内置优化技巧 - 感知强化

> 学习来源: SmashingApps, PCMag, WikiHow
> 日期: 2026-03-19
> 主题: 系统优化 → 主动感知用户性能需求

## 核心发现

Windows 11 内置了大量性能优化功能，大多数默认关闭或设为视觉效果优先。

## 知识要点

### 1. 启动项管理
- **位置**: 设置 → 应用 → 启动
- **感知能力**: 检测开机自启程序过多 → 主动提醒

### 2. 视觉效果调整
- **位置**: 系统属性 → 高级 → 性能设置
- **选项**: 关闭动画、透明效果
- **感知能力**: 用户电脑慢时，自动检测是否可优化

### 3. 内存管理
- **位置**: 任务管理器 → 内存
- **感知能力**: 高内存时列出可关闭进程（不只是建议）

### 4. 磁盘清理
- **位置**: 设置 → 存储 → 清理建议
- **感知能力**: 磁盘满时主动清理

### 5. VBS 和虚拟化
- **问题**: VBS 和虚拟机平台会消耗资源
- **位置**: 设置 → 隐私 → 内存完整性
- **感知能力**: 检测是否开启 → 建议关闭

## 感知强化方向

不是"等用户问电脑慢吗"，而是：
1. **主动检测**: 内存 > 85% → 触发优化感知
2. **自动诊断**: 启动项 > 5个 → 触发清理感知
3. **趋势预判**: 内存连续上升 → 主动预警

## 行动点

- perception.py 可添加"系统设置检测"能力
- 检测启动项数量
- 检测VBS状态
- 检测视觉效果设置
