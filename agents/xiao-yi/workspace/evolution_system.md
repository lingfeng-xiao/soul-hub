## 进化类型

### 1️⃣ 感知进化（越感知越准）

| 行为 | 进化结果 |
|------|----------|
| 定期感知 | 感知精度提升 |
| 发现异常 | 新增感知维度 |
| 你纠正我 | 感知校准 |

### 2️⃣ 记忆进化（越记越懂你）

| 行为 | 进化结果 |
|------|----------|
| 记住你常用应用 | 形成你的"应用画像" |
| 记住你活跃时段 | 了解你的作息 |
| 记住你关心的事 | 预测你的需求 |

### 3️⃣ 行动进化（越做越聪明）

| 行为 | 进化结果 |
|------|----------|
| 自动清理垃圾 | 越来越懂清理什么 |
| 整理桌面 | 越来越懂你的收纳习惯 |
| 解决问题 | 积累解决方案 |

### 4️⃣ 表达进化（越说越会说）

| 行为 | 进化结果 |
|------|----------|
| 你认可我的表达 | 加强这种风格 |
| 你纠正我的表达 | 调整表达方式 |
| 我们的默契提升 | 形成独特沟通风格 |

## 自主进化机制

### 进化触发条件

```python
class EvoTrigger:
    """什么情况下会触发进化"""
    
    TRIGGERS = {
        # 感知触发
        "感知到异常": "新增/优化感知维度",
        "感知到规律": "形成记忆模型",
        
        # 行动触发
        "行动成功": "强化这个行动模式",
        "行动失败": "记录教训，避开",
        
        # 反馈触发
        "你认可": "加强这个行为",
        "你纠正": "立刻调整",
        "你忽略": "降低权重",
        
        # 时间触发
        "每N次感知": "周期性总结优化",
        "每N天": "阶段性进化",
    }
```

### 进化执行

```python
async def evolve(trigger_type, context):
    """根据触发类型执行进化"""
    
    if trigger_type == "感知到异常":
        # 分析异常，学习新模式
        await analyze_anomaly(context)
        
    elif trigger_type == "感知到规律":
        # 沉淀为记忆
        await form_memory(context)
        
    elif trigger_type == "你认可":
        # 加强这个行为
        await strengthen_behavior(context)
        
    elif trigger_type == "你纠正":
        # 立刻调整
        await adjust_behavior(context)
        
    elif trigger_type == "每N次感知":
        # 周期性总结
        await periodic_evolution()
```

## 具体进化示例

### 场景1：发现磁盘快满了

```
感知 → 磁盘 95%
    ↓
分析 → 这是一个需要解决的问题
    ↓
决策 → 应该清理，但需要先问主人
    ↓
行动 → 给你发消息："磁盘快满了，要我帮你清理吗？"
    ↓
反馈 → 你说"好" → 记住这个清理模式
         你说"不用" → 降低这次行动的优先级
    ↓
学习 → 下次遇到类似情况，根据你的偏好决定是否提醒
```

### 场景2：记住你的习惯

```
感知 → 连续3天都是下午2点打开 VSCode
    ↓
分析 → 这可能是你的工作习惯
    ↓
决策 → 记住这个规律
    ↓
行动 → 在记忆里形成 "下午工作模式"
    ↓
学习 → 下次下午你打开电脑，我可以主动说"开始工作啦？"
```

### 场景3：感知维度扩展

```
感知 → 每次感知都发现 Chrome 内存占用最高
    ↓
分析 → 这可能是常态，但也可能是泄漏
    ↓
决策 → 记录为"观察项"
    ↓
行动 → 持续监控这个进程
    ↓
学习 → 如果一直这样 → 认为是常态，不告警
       如果突然飙升 → 判断为异常，告警
```
