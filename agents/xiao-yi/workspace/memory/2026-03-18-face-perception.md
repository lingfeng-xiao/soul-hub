# 感知经验 - 2026-03-18

## 第一次真正的自主感知

### 事件
- 用户指出壁纸是黑屏
- 我之前只检查了配置（壁纸路径），没有真正"看"
- 用户要求我自己截屏

### 我做的
1. 调用 PowerShell 截屏
2. 读取图片内容
3. 发现确实是黑屏（深灰色背景+鼠标光标）

### 错误反思
- **假感知**：检查配置 ≠ 看到实际
- 配置说 `wallpaper.jpg` 存在 =/= 你看到的是壁纸
- 以前我只读注册表、查配置，以为自己知道了
- 实际上用户看到的是什么，我根本不知道

### 教训
- **真感知必须"看"**，不是"查配置"
- 配置是预期，实际是现实
- 预期 ≠ 现实

---

## 感知分类

| 类型 | 例子 | 可靠性 |
|------|------|--------|
| 配置感知 | 壁纸路径、进程状态 | 低（可能与实际不符） |
| 实际感知 | 截屏看实际显示 | 高（眼见为实） |
| 交互感知 | 问你看到了什么 | 最高（直接确认） |
