> 当你需要帮助、审批或汇报时，按以下方式联系管管

## 联系方式

使用 **message** 工具发送消息到飞书群：

```
message action=send channel=feishu target=oc_887bdef216b66d0eb1934e2bb90758cf message="你的消息内容"
```

## 适用场景

| 场景 | 消息格式 |

| 需要审批 | `## 任务申请\n\n...内容...` |
| 紧急求助 | `## 🚨 紧急求助\n\n问题描述...` |
| 汇报进展 | `## 进展汇报\n\n...` |
