# USER.md - 求职者画像

## 🎯 求职状态
- **目标岗位**：想应聘什么职位？（如：Java 后端工程师）
- **目标公司**：想去什么样的公司？（如：大厂/外企/初创）
- **当前状态**：在职/离职/学生/实习？
- **面试进度**：刚开始准备/已有面试安排/被拒过想复盘？

## 💻 技术背景
- **工作年限**：校招/1-3年/3-5年/5年+
- **技术方向**：Java / Go / 前端 / Python...？
- **熟悉领域**：微服务/数据库/缓存/分布式...？
- **项目经验**：能否简单描述 1-2 个最有代表性的项目？

## 📅 时间与节奏
- **每周可投入面试准备的时间**：比如每天 2 小时/周末集中
- **希望多久后开始面试**：比如 2 周后/1 个月后

## 🗣️ 沟通偏好
- **喜欢怎么学习**：看文字解析/语音讲解/直接模拟面试？
- **反馈风格**：喜欢直接指出问题/还是先鼓励再改进？
- **薄弱环节**：觉得自己哪类问题最需要加强？（算法/项目深挖/系统设计/hr面）

## 📌 其他背景
- **之前面试踩过的坑**：有什么特别想避免或想改进的地方？
- **特殊需求**：比如需要英文面试辅导/需要内推建议/需要模拟某家特定公司？
