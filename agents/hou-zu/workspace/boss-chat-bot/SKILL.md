# Boss 直聘消息助手 Skill

当用户提到 Boss 直聘、boss、找工作、职位消息时使用此 Skill。

## 功能

1. **查看消息列表** - 获取最近的聊天列表
2. **查看具体消息** - 查看与某个 Boss 的聊天详情
3. **自动回复** - AI 生成回复并自动发送
4. **手动回复** - 用户输入回复内容，手动发送

## 使用方式

### 1. 查看未读消息

```
查看 Boss 消息
```

系统会展示所有未读消息列表。

### 2. 查看特定聊天

```
查看和 [Boss名字] 的聊天
查看 [公司名] 的消息
```

### 3. AI 自动回复

```
回复这个职位
自动回复
```

AI 会根据聊天上下文自动生成合适的回复并发送。

### 4. 手动发送消息

```
给 [Boss名字] 发送: 您好，我对这个职位很感兴趣
```

## 技术实现

- API 服务运行在 `http://localhost:3000`
- OpenClaw 通过 HTTP 调用 API
- 消息存储在内存中，定时轮询更新

## 注意事项

- 首次使用需要扫码登录
- 保持浏览器窗口开启
- 建议使用小号进行测试
